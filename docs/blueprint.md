# **App Name**: Cremonica

## Core Features:

- Vendor Onboarding: Super Admin creates and manages vendor profiles; regular vendors cannot register.
- Product Listing & Filtering: Browse products by category, location (Java), and price range with sort options (rating, popularity, price).
- Product Details: View detailed product information with image gallery, specs, and vendor contact via WhatsApp API.
- Authentication: User login and registration with redirection to login if cart actions are attempted while logged out.
- Shopping Cart: Add items to cart with automatic subtotal and grand total calculations.
- Checkout and Payment: Order summary with a mockup payment interface (or instructions for bank transfer).
- Admin Dashboard: Protected route for Super Admin to manage vendors and products.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) to evoke a sense of warmth and elegance.
- Background color: Light ivory (#FFFFF0) for a clean, minimalist feel, subtly complementing the beige.
- Accent color: Soft gold (#EEE8AA) to add a touch of luxury and highlight important elements, analogous to beige but slightly brighter.
- Body and headline font: 'Alegreya', a serif font to offer an elegant, intellectual, contemporary feel.
- Use 'Lucide React' icons for a clean and consistent aesthetic throughout the application.
- Mobile-first, responsive design with smooth transitions for a seamless user experience.
- Subtle animations for loading states and UI interactions to enhance user engagement.
